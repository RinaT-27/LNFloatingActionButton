## Requirements
- Swift 3.0 or later
- iOS 8.0 or later

## Installation
#### [Carthage](https://github.com/Carthage/Carthage)
```
github "akaimo/LNFloatingActionButton"
```

#### [CocoaPods](https://github.com/cocoapods/cocoapods)
```
pod 'LNFloatingActionButton', '~> 0.1'
```
